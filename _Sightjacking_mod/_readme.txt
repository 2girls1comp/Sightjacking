A mod that switches the game camera's point of view from the main character to the currently nearest NPC.

INSTALLATION:
- Copy "Sightjacking.dll" and "Sightjacking.ini" in your scripts folder (if you don't have one create it) in your GTA V root folder.

USAGE:
- Press J to toggle the mod ON/OFF
- To customize the key input edit the "Sightjacking.ini" file

REQUIREMENTS:
- Script Hook V
- Script Hook V Dot Net
